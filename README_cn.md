# DNA (Distributed Networks Architecture)

DNA是go语言实现的基于区块链技术的去中心化的分布式网络协议。

## 特性

* 可扩展的轻量级通用智能合约
* 跨链交互协议（进行中）
* 抗量子密码算法 (可选择模块)
* 中国商用密码算法 (可选择模块)
* 高度优化的交易处理速度
* 节点访问权限控制
* P2P连接链路加密
* 多种共识算法支持 (DBFT/VBFT)
* 可配置区块生成时间
* 可配置电子货币模型
