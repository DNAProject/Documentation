<h1 align="center">生成多签地址</h1>

DNA区块链客户端 `DNA-CLI` 提供了多签地址模块，用于根据公钥列表及签名门限 `M` 生成多重签名地址。可以在命令行中通过 `multisigaddr` 命令使用。

- `--pubkey`：用于指定多重签名地址的公钥列表，公钥之间用 `,` 分隔。
- `-m`：用于指定签名门限（默认为 `1`）。

要查看当前钱包账户中的公钥列表，使用 `dnaNode account list -v` 命令。

Pub key list:
